Comparação

A: Drawer B: Modal
UX Mobile ⭐ Nativo, gesto de swipe Bom, mas menos fluido
Recursão (material → sub-material) Empilha drawers ou navega ⭐ Empilha modais naturalmente
Espaço para conteúdo ⭐ Pode expandir até tela cheia Limitado pelo tamanho fixo
Fechar ⭐ Swipe para baixo Botão X ou backdrop
Contexto da lista ⭐ Lista visível parcialmente Lista coberta pelo overlay

⭐ Recomendação: Drawer (Bottom Sheet)

Para mobile-first, o drawer é o padrão nativo (Google Maps, Uber, iOS). O gesto de swipe é intuitivo, o conteúdo pode expandir se tiver muitos materiais, e a lista permanece parcialmente visível dando contexto. Para a recursão (clicar num sub-material), podemos empilhar drawers com um "← voltar" ou simplesmente trocar o conteúdo do drawer atual com animação de slide.
