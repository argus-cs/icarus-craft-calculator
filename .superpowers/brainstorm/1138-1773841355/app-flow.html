Fluxo do App & Componentes

Navegação e hierarquia de componentes Vue

Fluxo de Navegação
📋 Lista de Itens
busca + filtro por categoria
→
📄 Detalhe do Item
materiais + estações + qty
→
🔍 Modal Material
sub-materiais + qty + estação
Modal pode abrir outro modal (recursivo) — ex: Steel Ingot → Iron Ore + Carbon
Árvore de Componentes Vue
app.vue
pages/index.vue — página principal
SearchBar.vue — input de busca
CategoryFilter.vue — tabs: Ferramentas | Construções | Deployables
ItemList.vue — lista filtrada de itens
ItemCard.vue — card de cada item na lista
ItemDetail.vue — painel de detalhes (slide-up ou nova seção)
QuantityInput.vue — campo +/- quantidade
MaterialRow.vue — linha de material (clicável)
StationBadge.vue — badge da estação de craft
MaterialModal.vue — modal de detalhe do material
QuantityInput.vue — reutilizado
MaterialRow.vue — reutilizado (permite recursão)
StationBadge.vue — reutilizado
Composables (lógica reutilizável)
composables/
useItems.ts — carrega items.json, busca, filtro por categoria
useCraftCalculator.ts — cálculo de materiais × quantidade
useStations.ts — lookup de estações por ID
